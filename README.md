## 🛠️ Troubleshooting

### App does not start

Try these steps:

1. Close the app.
2. Right-click it and run it again.
3. Make sure Windows allowed the app to run.
4. Check that virtualization is enabled in BIOS or UEFI.
5. Restart your PC and try again.

### Setup stops early

Try this:

1. Confirm you are online.
2. Make sure there is enough free disk space.
3. Open the app again and repeat the setup.
4. If a console window shows an error, read the last line and try again.

### Terminal opens, then closes

Try this:

1. Start the app from the main window if one exists.
2. Do not double-click the file several times.
3. Wait for the setup to finish before closing the terminal.

### Commands fail to run

Try this:

1. Open the shell again.
2. Check that the environment is still active.
3. Run the command from inside the app, not from normal Windows Command Prompt.
4. Update the package list first, then try again.
